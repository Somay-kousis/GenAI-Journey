
# Day 0 — Introduction to LangChain

## What is LangChain?

LangChain is an open-source framework for building applications powered by Large Language Models (LLMs). 

Think of it as:

```text
Python + AI Components
        ↓
     LangChain
        ↓
   AI Applications
```

---

# Why Do We Need It?

Without LangChain:

```text
Prompt
 ↓
LLM API
 ↓
Output
```

Easy for small projects.

---

For bigger projects:

```text
PDFs
Websites
Memory
Chat History
Databases
Tools
Agents
```

Managing everything manually becomes painful.

---

LangChain provides:

```text
Models
Prompts
Chains
Memory
RAG
Agents
```

inside one ecosystem. 

---

# Main Benefits

### Model Agnostic

```text
GPT
Claude
Gemini
Llama
Mistral
```

Same framework.

Switch models with minimal code changes.

---

### Chains

Connect multiple AI steps.

```text

 ↓

 ↓

 ↓

```

---

### Memory

Allows chatbots to remember conversations.

```text

 ↓

 ↓
Remembers Context
```

---

### Complete Ecosystem

Everything under one roof.

```text

Prompts
RAG
Agents
Memory
```

# One Line Summary

```text
LangChain is not an AI model.

It is the framework that helps developers build AI applications using AI models.
```
